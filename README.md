SPP
===

Lisp interpreter written in C.

Compiling
---------

GNU/Linux & other Unix systems using GCC:

<i>cd SPP; make</i>

or you can use the Code::Blocks project.

Windows:

Code::Blocks project should probably work, even through I haven't tested it yet.
